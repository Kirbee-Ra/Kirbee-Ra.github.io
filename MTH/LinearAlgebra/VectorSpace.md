# 체
체(Field, $$\mathbb{F}$$)란 교환성, 결합성, 항등원, 역원, 분배성을 만족하는 곱셈과 덧셈을 가진 집합입니다. 물리학에서는 주로 실수($$\mathbb{R}$$)나 복소수($$\mathbb{C}$$)가 스칼라의 체로 쓰입니다.
